# HTML Pages Examples

This template uses `@web-widget/html` as the primary rendering framework, with tagged template literals for server-side HTML rendering. React widgets are integrated for interactive islands.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/web-widget/web-widget/tree/main/examples/html)

## Development

```shell
pnpm install
pnpm dev
```

## File Description

- `routes/**/*@route.html.ts` Route module using HTML tagged templates, runs only on the server side
- `routes/**/*@middleware.*` Middleware, runs only on the server side
- `routes/**/*@widget.*` User-interactive components (React), run on both server and client
- `entry.client.ts` Client entry
- `entry.server.ts` Server entry
- `importmap.client.json` Import map configuration for the client
- `routemap.server.json` Routing configuration file, auto-generated by development tools
